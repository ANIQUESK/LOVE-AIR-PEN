🪄 What is Love Air Pen?

Love Air Pen is an AI-powered gesture drawing system that lets you write in the air using your finger.

Using computer vision, your hand becomes a neon glowing pen, allowing you to draw or write without touching the screen.

It combines technology, creativity, and emotion into a single interactive experience.

💡 Key Features
🖐 Gesture Writing

Write in the air using index finger pinch gesture detection.

✍ Ultra Smooth Neon Ink

Custom stroke stabilization algorithm provides:

• smooth curves
• stabilized motion
• neon glowing strokes

💖 Love Themed UI

The interface includes:

• dark pink background
• glowing neon strokes
• floating hearts
• romantic aesthetic theme

🧠 AI Hand Tracking

Powered by:

MediaPipe Hands

Tracks 21 hand landmarks in real-time.

🔤 AI Handwriting Recognition

Includes a module:

ai/handwritingAI.js

This allows recognition of handwritten text using OCR/AI techniques.

📁 Export Drawings

Save your creation as:

• PNG
• SVG
• PDF

5️⃣ Start Writing

🎯 How It Works

1️⃣ Camera captures your hand
2️⃣ MediaPipe detects 21 hand landmarks
3️⃣ System tracks the index finger position
4️⃣ Pinch gesture triggers drawing
5️⃣ Motion becomes smooth neon strokes on canvas
